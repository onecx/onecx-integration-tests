= Getting Started: Prerequisites and First Run

This page is the entry point for new users. It lists the required tools, walks through the first-run checklist, and provides role-based reading paths that direct each audience to the most relevant chapters. Complete the checklist before consulting any other page.

== Contract

Start with the configuration contract before running anything.

See xref:contract.adoc[Contract and Configuration] for:

* `platform.json` structure
* Required and optional fields
* Container definitions for service, BFF, UI, and E2E
* Validation and fail-fast behavior

== Role-Based Paths

Choose your path based on your role:

* Integrator: xref:contract.adoc[Contract and Configuration], then xref:e2e.adoc[E2E Testing]
* Test engineer: xref:e2e.adoc[E2E Testing], then xref:reference.adoc[Reference]
* Platform maintainer: xref:architecture.adoc[Architecture], then xref:ci.adoc[CI and GitHub Actions]
* UI repository maintainer: xref:ui-repository.adoc[UI Repository Integration], then xref:contract.adoc[Contract and Configuration]

== First Run Checklist

Complete these steps in order:

* Install Node.js, npm, and Docker
* Run `npm install`
* Prepare `integration-tests/platform/platform.json`
* Run `npm run it:run -- --dry-run`
* Run `npm run it:run -- --verbose --capture-logs`
* Inspect `integration-tests/artifacts/<run-id>/summary.json`
